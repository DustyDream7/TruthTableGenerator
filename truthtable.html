Truth Table Generator

This page contains a JavaScript program which will generate a truth table given a well formed formula of sentential logic. Enter multiple formulas separated by commas to include more than one formula in a single table. You can choose between a full truth table, a table that only displays the column underneath the main connective, a table in plain text, and a table formatted for LaTex. (See here for a program to produce formulas involving only the Sheffer Stroke.)

Symbols
Negation~
Conjunction&
Disjunctionv
Conditional>
Biconditional< >
Sheffer Stroke|
Absurdity#
Examples:
~A
(A & B)
(# > (B v ~A))
(A & (~B > C))
(A<>(BvC)), A, (~B>C)
